HTML slides you can edit by clicking

This is a plain, self-contained HTML deck — exactly what Claude Code generates. Open it in LiveDeck and edit any text directly, like PowerPoint.

The problem

  • Generating a deck is easy — iterating on it is not.
  • You either re-prompt the model or hand-edit raw HTML.
  • There's no intuitive, what-you-see-is-what-you-get way to nudge a word.

What LiveDeck adds

01

Click to edit

Click any heading, bullet, or paragraph and type. No syntax.

02

Stays plain HTML

Your edits save straight back to the same file Claude can read.

03

Live two-way sync

Re-prompt in Claude Code and watch the slide update instantly.

Try it now
